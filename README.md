# Projet Philosophes

## Composition du groupe
- Vincent Renard
- Romain Guidez

## Premier algorithme
Les philosophes possèdent trois états :
* penser (T)
* avoir faim (H)
* manger respectivement (E).
Les canaux possèdent trois états :
* aucune flèche N
* flèche vers la gauche L
* flèche vers la droite respectivement R.
